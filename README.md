# 🧠 LLM Playground

An interactive Streamlit application for experimenting with Large Language Models (LLMs). Explore model behavior by adjusting inference parameters, maintaining conversation context, and comparing responses across multiple Groq-hosted models.

## ✨ Features

- 🤖 Multi-LLM Support
- 💬 Conversation Memory
- 🌡️ Temperature Control
- 🔢 Max Token Configuration
- ⚙️ Model Selection
- 🚀 Fast Inference with Groq API
- 🎨 Clean Streamlit Interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- python-dotenv

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
