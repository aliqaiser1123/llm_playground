# 📌 Overview

LLM Playground is a beginner-friendly application built to explore the behavior of Large Language Models through real-time experimentation. Users can interact with multiple LLMs, adjust inference parameters, and observe how these settings influence generated responses.

This project was developed as part of my AI Engineering learning journey to gain practical experience with LLM APIs and prompt-based interactions.

---

# ✨ Features

- 🤖 Multiple LLM Support
- 💬 Multi-turn Conversation Memory
- 🌡️ Temperature Control
- 🎯 Max Token Configuration
- 🔄 Model Selection
- 📝 Interactive Chat Interface
- ⚡ Fast Inference using Groq API
